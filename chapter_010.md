# Начало работы

## Задача 0

Создайте файла `README.md` в своей подпапке содержащий следующий текст

```text
# Фамилия Имя Отчество
Вариант V
```

`Фамилия Имя Отчество` замените вашими реальными данными, `V` - номером вашего варианта.
